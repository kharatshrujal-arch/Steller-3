# Assets Directory

This directory contains all media assets for the AidPulse project documentation.

## 📁 Structure

```
assets/
├── screenshots/     # UI screenshots for README
│   ├── UI1.png     # Dashboard interface
│   ├── UI2.png     # Aid case creation
│   └── UI3.png     # Real-time event stream
└── videos/         # Demo videos
    └── demo.mp4    # Full application walkthrough
```

## 📸 Screenshots

Located in `screenshots/`:
- **UI1.png** - Main dashboard showing wallet connection and balance
- **UI2.png** - Aid case creation interface
- **UI3.png** - Live event stream ledger

## 🎥 Videos

Located in `videos/`:
- **demo.mp4** - Complete demo walkthrough of AidPulse features

## 📝 Usage

These assets are referenced in the main [README.md](../README.md) to provide visual documentation of the application's features and functionality.

### Adding New Assets

**Screenshots:**
1. Place PNG/JPG files in `screenshots/`
2. Use descriptive names (e.g., `feature-name.png`)
3. Reference in README: `![Description](assets/screenshots/filename.png)`

**Videos:**
1. Place MP4/WebM files in `videos/`
2. Use descriptive names (e.g., `feature-demo.mp4`)
3. Reference in README: `[Watch](assets/videos/filename.mp4)`

## 🎨 Guidelines

- **Format**: PNG for screenshots, MP4 for videos
- **Size**: Keep files under 10MB when possible
- **Naming**: Use lowercase with hyphens (kebab-case)
- **Quality**: High-res screenshots, 1080p minimum for videos
- **Compression**: Optimize images before committing

---

**Note**: These assets are committed to the repository to ensure documentation is complete. Large video files should be hosted externally for production repos.
