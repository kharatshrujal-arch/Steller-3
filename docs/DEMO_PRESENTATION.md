# Demo Presentation

## Slide 1: Problem

Community aid groups need a simple way to prove that donated funds were sent and released to the intended beneficiary.

## Slide 2: Solution

AidPulse connects a testnet wallet, shows balance, sends XLM, and uses a Rust Soroban escrow contract for transparent aid cases.

## Slide 3: User Flow

Connect Freighter, check balance, send testnet XLM, call the aid case contract, and watch live events.

## Slide 4: Technical Stack

Rust, Soroban, Stellar Testnet, Freighter, Albedo, React, TypeScript, Horizon, Soroban RPC, and GitHub Actions.

## Slide 5: Smart Contracts

AidPulse handles case creation, funding, release, authorization, token transfer, and events. Reputation receives an inter-contract score update.

## Slide 6: Quality

The project includes contract tests, frontend tests, clear error handling, loading states, mobile responsive design, CI, deployment workflow, and documentation.

## Slide 7: Future Work

Add case creation in the UI, beneficiary verification, notification webhooks, and indexed event history.
